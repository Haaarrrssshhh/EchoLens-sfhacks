Keyboard Shortcuts

  • Option+Shift+A - Toggle screen reader (Mac)
  • Alt+Shift+A - Toggle screen reader (Windows/Linux)
  • Tab - Navigate through interactive elements
  • Arrow Up/Down - Navigate with screen reader
  • Space/Enter - Activate focused element
  • Alt+H / Option+H - Read headings
  • Alt+L / Option+L - Read landmarks
  • Alt+P / Option+P - Read paragraphs
  • Alt+I / Option+I - Read images
  • Escape - Stop reading

When screen reader is active, use arrow keys to navigate through all elements, including images and paragraphs.
